The Quick Bucks Application

This is a Full stack application, using Java with Hibernate. Using Maven for project dependencies. The front end is using Bootstrap for styling and Javscript to conducting fetch requests. In this application you can view all reimbursements, add your own reimbursement, view pending reimbursements to be resloved, and there is also an option to see all users currently in the system. There are some features that are disabled until they roll out to the application. This app is still in beta.
